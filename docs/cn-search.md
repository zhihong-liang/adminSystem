# 搜索组件

## 组件地址

`src/components/cn-page/CnSearch.vue`

## 功能描述

基于 [CnForm](./cn-form.md) 组件，实现通用列表查询页面的条件搜索功能。

## 参数

|属性|说明|类型|必选|可选值|默认值
|:--|:--|:--|:--|:--|:--
|onSearch|搜索方法，`CnPage` 组件内部实现此方法|`function`|否|-|-
|onReset|重置方法，`CnPage` 组件内部实现此方法|`function`|否|-|-

其余属性参考 [CnForm](./cn-form.md)。
